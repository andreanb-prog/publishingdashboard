Claude + ChatGPT

Want to improve your craft? This is how Claude or ChatGPT can help you identify exactly where your chapters are working — and where they're not.

Use these prompts as a starting point, then make them yours. Add dimensions that matter for your genre. Remove ones that don't. The sharper you customize, the sharper the diagnostic.

One recommendation: once you run a prompt and get output you like, save the customized version in the Claude Project or chat thread for that book. That way you can run it again on the next draft without starting over — and without coming back here (though you're always welcome).

ChatGPT note: This version includes a stricter formatting constraint at the top. If ChatGPT still adds unsolicited commentary, reply with: "Follow the format exactly. Return only what was asked." Use GPT-4o for best results — GPT-3.5 may struggle with longer manuscripts in Mode 1.
